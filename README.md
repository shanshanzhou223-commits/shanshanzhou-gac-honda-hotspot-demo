# 广本车型 × 热点话题匹配 Demo

一个基于 Streamlit 的交互式演示工具，用于向客户展示：

- 车型形象标签库（A库）
- 热点话题六维标签体系（B库）
- 自动为输入热点打标签
- 车型与热点的匹配评分算法
- 推荐等级（S/A/B/C）
- 内容角度建议

## 快速开始

### 1. 安装依赖

```bash
cd /Users/sancoo/gac_honda_hotspot_demo
pip install -r requirements.txt
```

### 2. 运行 Demo

```bash
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`。

## 功能说明

| Tab | 功能 |
|-----|------|
| 🏠 车型标签库 | 展示所有车型 A库标签 + 车型图 |
| 🔥 平台热点池 | 按平台（抖音/微博/小红书/今日头条）展示 Top10 热点 |
| 🎯 单热点全车型匹配 | 输入任意热点，自动打标并一次性匹配全部车型，输出雷达图与内容角度 |

## 车型图片处理

把车型原图放入 `images/` 文件夹，命名为 `{车型名}.png`（如 `雅阁.png`、`P7.png`），然后运行：

```bash
python3 process_images.py
```

脚本会自动：
- 统一裁剪为 4:3 尺寸
- 根据图片本身色调生成填充背景，使各车型图在页面上更整体和谐

处理后的图片保存在 `images/processed/`，Demo 会优先读取该目录。

## 评分模型

- **品牌契合分** = 直接功能 30% + 价值观/叙事 25% + 人群兴趣 25% + 竞品/品类 15% + 文化时刻 5%
- **完整契合分** = 热度 25% + 品牌契合 35% + 情绪共鸣 20% + 传播可行性 10% + 安全分 10%
- **推荐等级**：S级（≥85）、A级（70-84）、B级（55-69）、C级（<55）
